# MediaPipe Experiments

## Descripción
Repositorio dedicado a la experimentación, pruebas de concepto y desarrollo de pipelines de visión computacional utilizando [Google MediaPipe](https://developers.google.com/mediapipe). El enfoque principal de este proyecto es la optimización del rendimiento y la integración eficiente de modelos de machine learning en tiempo real.

## Arquitectura del Sistema
El proyecto está estructurado para aislar las dependencias y mantener un entorno de ejecución limpio. Se recomienda encarecidamente el uso de entornos virtuales para evitar colisiones a nivel de sistema.

## Requisitos Previos
* **Python**: `3.9` o superior.
* **Gestor de paquetes**: `pip` o `uv`.
* Dependencias base del sistema para el manejo de cámaras (ej. `v4l2` en entornos Linux).
